# Niffler

A view of the tumblr dashboard, in a native application window, using *electron*

## Usage

- Requires the following tumblr api tokens and secrets
```
CONSUMER_KEY
CONSUMER_SECRET
TOKEN
TOKEN_SECRET
```

- Put the above values in a `.env`
- Finally
```
$ npm i
$ npm run start
```
